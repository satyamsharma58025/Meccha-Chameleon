# IP Safety Checklist

Complete **every item** before any public build, store listing, or marketing material.

- [ ] Game name, logo, and store listing text are entirely original — no phonetic variants of existing trademarked genre names
- [ ] Mascot/character design is your own species/concept (**Claylings** — clay golem, not chameleon/lizard)
- [ ] All maps, props, textures, and UI icons are custom-built, not traced or asset-ripped
- [ ] Mode names, menu labels, and UI copy are original wording
- [ ] Marketing screenshots/trailers show only original assets
- [ ] Soundtrack and SFX are licensed, original, or properly-attributed royalty-free
- [ ] Store description does not reference or compare to any specific existing title by name

## Do Not Use

- Chameleon/reptile mascot theming
- Phonetic clones of existing genre titles
- Copied map layouts traced from reference footage
- Side-by-side comparison marketing vs. any existing game

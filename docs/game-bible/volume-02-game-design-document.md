# Volume 2 — Complete Game Design Document

## 1. Game Overview

This game is an original multiplayer party title built around camouflage, observation, and deception. Each match is a short contest of visual adaptation and pursuit.

## 2. Core Gameplay Loop

1. Players enter a lobby.
2. Roles are assigned.
3. Hiders paint and reposition during prep.
4. Seekers attempt to locate hidden players in hunt.
5. The round resolves via timer or full clear.

## 3. Core Mechanics

- Sampling surface color and material approximations
- Applying paint to the player avatar
- Switching poses and using geometry for cover
- Tagging and inspecting opponents
- Using short taunts to create scoring opportunities

## 4. Game Modes

- Classic
- Overrun
- Trade Places
- Lone Wolf

## 5. Content Pillars

- Fast rounds
- Clear visual feedback
- Skill-based but approachable camouflage
- Strong local multiplayer testability

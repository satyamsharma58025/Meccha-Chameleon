# Volume 13 — QA, Testing & Performance

## 1. Test Layers

- Unit tests for scoring and validation logic
- Integration tests for state transitions
- Device smoke tests for touch and keyboard input

## 2. Performance Targets

- 60 FPS mid-range mobile
- Low allocation rate in input and paint hot paths
- Stable frame pacing during paint and camera movement

## 3. QA Focus Areas

- Round flow reliability
- Paint visibility and feedback
- Tag and inspect interactions
- Accessibility and input clarity
